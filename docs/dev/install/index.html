Installing odinson-rest (For Development)

Requirements

  • docker
  • JDK 11
  • sbt
  • 2G of RAM

Install

Clone the odinson-rest repository.

Running

To run the Odinson REST API in development mode, run the following command:

sbt web

Open your browser to localhost:9000.
